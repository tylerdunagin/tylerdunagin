# Tyler Dunagin

I build AI-assisted operating systems for essential-service businesses.

Not software companies. Actual field operations — multifamily turns,
water-loss risk, EFI calibration shops, career intelligence platforms.
The common thread: industries where the work is physical, the margins
are tight, and the data infrastructure is basically nonexistent.

Most of what's pinned here I built to run inside businesses I operate
or own. Some of it started as a solution to a problem I couldn't find
anyone else solving. All of it is in production or pilot-ready.

I'm based in Northeast Ohio. Three sons. I build cars with my dad.
